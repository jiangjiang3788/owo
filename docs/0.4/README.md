# OWO Memory Brain v0.4

v0.4 的主线是“历史大整理”。

v0.3 已经让新记忆脑拥有事件、事实、家族、graph、长期模型、注入预览、调度和记忆小屋。v0.4 不直接正式接管聊天，而是先让新记忆脑能够读完整个过去。

## v0.4 路线

```text
v0.4.0 历史源扫描器
v0.4.1 历史切片 / 游标
v0.4.2 回填队列 / 断点续跑
v0.4.3 历史事件回填
v0.4.4 历史事实回填
v0.4.5 去重 / 冲突 / 过时事实
v0.4.6 全量家族 / graph 重建
v0.4.7 全历史长期模型重建
v0.4.8 新旧注入对照 / 接管演练
v0.4.9 单一 owner 切换门
```

## 边界

到 v0.9 完成之前，Memory Brain 仍然是可读取、可整理、可预览的新脑；正式 prompt 注入继续由当前选择的旧记忆 owner 执行。

v0.4.0 只扫描、统计、建索引；v0.4.1 只生成 archiveChunks 和 archiveCursors；v0.4.2 只生成 backfillJobs / backfillRuns 和断点状态。三版都不总结、不迁移、不跑 AI、不改旧记忆。

完整 v0.4～v0.8 计划见 `docs/0.4/roadmap-v0.4-v0.8.md`。
