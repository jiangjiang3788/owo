## 分卷原则

| 文档类型 | 放置位置 | 说明 |
|---|---|---|
| 历史架构 gate / V2～V38.1 | `docs/0.1` | V 编号继续作为 gate 编号存在，但不再作为产品版本号。 |
| 当前功能 release / v0.2.x | `docs/0.2` | 新增 release 文档优先放这里。 |
| gate 固定入口 | 根路径 `docs/*.md` | 例如 `docs/smoke-memory.md`、`docs/css-ownership.md`，为了工具兼容继续保留。 |
| 根路径固定入口 | `docs/release-plan.md` | 只保留总入口，不再新增根路径 `release-v0.2.x-plan.md`。 |

## 后续建议
