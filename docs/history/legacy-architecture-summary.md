# 旧架构整改历史摘要

本文件合并原 `docs/0.1` 与 `docs/0.2` 中的大量逐版本计划。原文档主要记录了以下阶段：

- 存储单写者、Dexie Adapter、加载修复与备份适配；
- AI Provider 配置、请求 Adapter 与消息语义拆分；
- Chat Prompt、渲染、设置页、档案、向量、日记和世界书的切片迁移；
- Forum、Theater、Peek、Video Call 等功能边界；
- Screen Registry、CSS Ownership、Legacy Globals Gate。

仍会被自动 Gate 读取的文档已保留在 `docs/operations/gates/`。其余逐小版本计划已删除，避免历史计划继续被误当成当前架构说明。

当前架构以 `docs/architecture/architecture.md` 和 `docs/releases/roadmap.md` 为准。
